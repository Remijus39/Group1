## Project Description
The **Chat App** is a Flutter-based mobile application that allows users to communicate in real-time using Firebase as the backend. Users can register, log in, and chat with other users securely. The app includes push notifications for new messages, profile management, and a clean, organized UI.  

This project demonstrates teamwork with 10 members, each contributing to different parts of the application, and shows proper use of Flutter, Firebase, and GitHub collaboration for submission.

## Features
- **User Authentication**: Secure login and registration using Firebase Authentication.  
- **Real-time Chat**: Users can send and receive messages instantly via Firestore.  
- **Push Notifications**: Notifications alert users of new messages.  
- **User Profile Management**: Users can view and update their profile information.  
- **Organized Assets & Images**: Includes app logos, backgrounds, and icons.  
- **Responsive & Consistent UI**: All screens are designed for a smooth user experience.  
- **Documentation & Testing**: Fully tested features and clear project documentation.
